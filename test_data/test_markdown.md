# EvoIndex-2.0 测试文档

## 项目概述

EvoIndex-2.0 是一个智能知识库索引系统，支持：

1. **层次化树索引** - 保留文档逻辑结构
2. **多模态解析** - 支持 Markdown 和 PDF
3. **自动意图识别** - 4 种意图分类
4. **自进化学习** - 持续优化检索质量

## 核心特性

### 层次化树索引

传统的 RAG 系统使用 flat chunks，丢失了文档的层次结构。EvoIndex-2.0 使用树索引，保留了章节、子章节的逻辑关系。

### 多模态解析

- ✅ Markdown - 完美支持
- ✅ PDF - 基于 PyMuPDF
- ✅ 自动检测文档类型

### 意图识别

系统可以识别 4 种查询意图：

1. **精确匹配** - 直接关键词查询
2. **语义搜索** - 理解查询意图
3. **聚合查询** - 跨章节汇总
4. **关系查询** - 概念关联分析

## 使用方法

### 构建索引

```bash
node src/core/tree.mjs document.md index.json
```

### 查询索引

```bash
node src/core/query_tree.mjs "如何使用 EvoIndex-2.0？" index.json
```

## 性能指标

- **构建耗时**: <1ms/100章节
- **查询耗时**: ~2-6秒（真实 LLM）
- **准确率**: 85%+（人工评估）

## 总结

EvoIndex-2.0 是下一代智能知识库系统，结合了层次化索引、意图识别、自进化学习等多项先进技术。
