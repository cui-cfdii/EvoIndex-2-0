## 🔬 算法逻辑

### 核心流程

```
初始化
  ↓
For i = 1 to 50:
  1. 随机选择样本
  2. jieba 分词
  3. LLM 评估（带缓存）
  4. 检查回滚条件
  5. 记录历史
  6. 检查阶段进展
  7. 达到稳定后降频
  ↓
输出结果
```

### 关键机制

#### 1. 阶段性进化
```javascript
stageThresholds: [0.7, 0.8, 0.85, 0.9, 0.93]

达到阈值 → 进入下一阶段
达到 0.9 → 降低进化频率（每 5 轮进化 1 次）
```

#### 2. 回滚保护
```javascript
rollbackThreshold: 0.05

if (currentScore < bestScore - 0.05) {
  rollback();  // 回滚到最佳状态
}
```

#### 3. 缓存优化
```javascript
evalCache: Map (容量 1000)

async evaluate(text, tokens) {
  if (cache.has(key)) return cache.get(key);
  const result = await llm.evaluate(...);
  cache.set(key, result);
  return result;
}
```
