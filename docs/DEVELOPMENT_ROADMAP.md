## 🎯 项目愿景

打造**懂用户 + 懂知识**的智能知识管家，实现：
- ✅ 理解用户意图（历史回忆、知识查询、术语解释）
- ✅ 个性化检索（偏好领域优先）
- ✅ 自进化学习（遇到不会的自动学）
- ✅ 双记忆协同（MEM0 个人记忆 + EvoIndex 知识记忆）

**降级方案**（如果 MEM0 API 不可用）:
```javascript
class MemoryService {
  constructor() {
    try {
      this.mem0 = new Mem0Service();
      this.mode = 'mem0';
    } catch (error) {
      this.local = new LocalMemoryService();
      this.mode = 'local';
    }
  }

  async storeQuery(query, intent, results) {
    if (this.mode === 'mem0') {
      await this.mem0.storeQuery(query, intent, results);
    } else {
      await this.local.appendToFile('memory/query_history.md', {
        query, intent, results,
        timestamp: new Date().toISOString()
      });
    }
  }
}
```

**时间**: 2026-03-19（Phase 2 开始前）

**选项**:
- A. API 可用 → 使用 MEM0
- B. API 不可用 → 使用降级方案（本地记忆）

**影响**:
- A: 开发时间 +2 天，功能更强大
- B: 按原计划，功能略简化
