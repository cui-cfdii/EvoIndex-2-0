## 🎯 核心技术亮点

### 1. 意图识别（4 种类型）

```javascript
// 知识查询
"肺结核的治疗方案有哪些" → knowledge_query, medical_ai

// 历史回忆
"我上次查的肺结核治疗方案呢？" → history_recall

// 术语解释
"什么是 RAG？" → term_explanation, llm

// 领域探索
"介绍一下金融科技" → domain_exploration, fintech
```

### 2. 自进化触发器（5 种触发类型）

```javascript
// 低匹配度触发
matchRate < 0.8 → LOW_MATCH

// 新领域触发
domain not in interests → NEW_DOMAIN

// 频繁查询触发
query_count >= 3/week → FREQUENT_QUERY

// 用户兴趣触发
confidence >= 0.5 → USER_INTEREST

// 低召回率触发
recall < 0.92 → LOW_RECALL
```

### 3. 降级策略

```javascript
// 自动检测与降级
if (mem0Available) {
  mode = 'mem0';
} else {
  mode = 'local';  // 自动降级
}
```

## ⚠️ 已知问题

### 1. 实体提取准确率

**问题**: 简单分词导致实体提取不准确  
**影响**: 部分复杂查询的实体识别失败  
**解决方案**: 引入 jieba 分词或 NLP 模型

### 2. MEM0 API 集成待完成

**问题**: MEM0 API 连接测试和调用未实现  
**影响**: 当前只能使用本地记忆模式  
**解决方案**: 确认 MEM0 API Key 有效性后实现

### 3. 结果融合算法简单

**问题**: 当前融合算法只是简单拼接  
**影响**: 融合效果不够智能  
**解决方案**: 实现基于权重的融合算法
